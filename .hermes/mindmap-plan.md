# 思维导图功能 — 设计计划

## 决策汇总

| 决策 | 结论 |
|------|------|
| 数据目录 | `mindmaps/`，独立于笔记 |
| 节点结构 | `{ id, text, children[], collapsed }`，纯文本节点 |
| 节点内容 | 每个节点正文就是 `text` 字段，不需要额外富文本备注 |
| 排序 | 按创建顺序，不支持拖拽 |
| 存储格式 | 每个导图一个 `.json` 文件 |
| 后端 API | `list_mindmaps` / `create_mindmap` / `save_mindmap` / `delete_mindmap` |
| i18n | 导航栏"备用"→"导图" / "Mindmap"，UI 文案加入 i18n |
| 暗色模式 | 复用现有 CSS token |

## 数据结构

```json
{
  "id": "mindmap-1712345678000",
  "title": "项目计划",
  "updatedAt": 1712345678000,
  "root": {
    "id": "n1",
    "text": "根节点",
    "collapsed": false,
    "children": [
      { "id": "n2", "text": "前端", "collapsed": false, "children": [] },
      { "id": "n3", "text": "后端", "collapsed": true, "children": [
        { "id": "n4", "text": "API", "collapsed": false, "children": [] }
      ]}
    ]
  }
}
```

## 交互设计

```
┌─ 导图列表（左侧边栏）──────────┬─ 导图画布（右侧）──────────┐
│ [+新建]                        │                            │
│                                │  🟊 根节点                 │
│ ● 项目计划                     │    ├─ 🟊 前端              │
│   学习路线                     │    └─ 🟊 后端 [▸]        │
│                                │         └─ API           │
│                                │                            │
│              [🗑 回收站]       │ [+ 添加子节点] [✕ 删除]  │
└────────────────────────────────┴────────────────────────────┘
```

操作：
- 点击 🟊 展开/折叠子节点
- 点击节点文本 → 进入编辑模式（原地编辑）
- 点击 [+ 添加子节点] → 在当前选中节点下新增子节点
- 点击 [✕ 删除] → 删除当前节点及其子树
- 新建导图 → 自动创建一个根节点"未命名导图"
